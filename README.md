# AI-Driven Financial Intelligence and Decision Support System

## 📌 Overview

This project is a simple AI-powered financial analysis system that helps businesses understand their data and make better decisions.

It takes financial data (like sales, revenue, and profit) and provides:

- Future revenue predictions  
- Detection of unusual financial patterns  
- Risk analysis  
- Simple business recommendations  

The goal is to move from **basic reporting** to **intelligent decision support**.

---

## 🚀 Features

### 📊 1. Financial Forecasting
- Predicts future revenue using time-series modeling
- Shows expected trends for the next 30 days

### 🚨 2. Anomaly Detection
- Detects unusual spikes or drops in revenue
- Helps identify potential issues or opportunities

### ⚠️ 3. Risk Detection
- Identifies declining revenue trends
- Detects rising expenses
- Classifies risk levels (Low / Medium / High)

### 🔁 4. Scenario Simulation
- Simulate increase/decrease in expenses
- Understand impact on profit

### 💡 5. Insights & Recommendations
- Provides simple business suggestions
- Easy to understand for non-technical users

---

## 🛠️ Technologies Used

- Python  
- Pandas  
- Matplotlib  
- Prophet (for forecasting)  
- Streamlit (for dashboard)
